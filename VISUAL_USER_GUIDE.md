## Basic Verse Lookup

### Step 1: Enter a Verse Reference

Simply type a Bible reference in the input box. LiveVerses understands many formats!

![Entering Verse](screenshots/02-entering-verse.png)

**Supported Formats:**
- `John 3:16` - Full book name
- `Jn 3:16` - Abbreviated

4. **Keep the Control Window private** - share only Display Window
5. **Practice with abbreviations** for speed (Jn instead of John)

## Troubleshooting

### Verses Not Appearing

**Problem:** Verse not found
**Solution:**
- Check spelling of book name
- Verify chapter and verse exist
- Try full book name instead of abbreviation
- Ensure Bible data files are loaded (check browser console)

### Display Window Not Opening

**Problem:** Can't open display window
**Solution:**
- Allow popups in browser settings
- Try a different browser (Chrome recommended)
- Check for popup blocker extensions

### Display Window Not Updating

**Problem:** Control changes don't appear in display
**Solution:**
- Check connection status (should be green)
- Close and reopen Display Window
- Refresh both windows
- Ensure both windows are from the same source
