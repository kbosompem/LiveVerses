## How to Run the App

### Option 1: Double-Click (Easiest!)
1. **Simply double-click `index.html`**
2. It will open in your default browser
3. Start using immediately!

### Option 2: Open from Browser
1. **Right-click `index.html`**
2. **Select "Open With"** ‚Üí Choose your browser (Chrome recommended)
3. Done!

### Option 3: Drag and Drop
1. **Drag `index.html`** into your browser window
2. The app opens instantly

### Option 4: Command Line (Mac/Linux)
```bash
open index.html
```

Or on Windows:
```cmd
start index.html
```

### Option 5: Local Web Server (Advanced)
If double-clicking doesn't work:

```bash
# Start a simple web server
python3 -m http.server 8000

# Then open browser to:
# http://localhost:8000
```

## Troubleshooting

### Browser Shows Error or Blank Page?

**Try These Solutions:**

1. **Use a modern browser**
   - Chrome (recommended)
   - Edge
   - Safari
   - Firefox

2. **Allow local file access**
   - Some browsers block local files
   - Use the web server method instead (Option 5 above)

3. **Check browser console**
   - Press F12 (or Cmd+Option+I on Mac)
   - Look for errors in the Console tab

4. **Verify data files exist**
   ```bash
   ls -lh data/
   ```
   Should show:
   - kjv.json (4.3 MB)
   - asv.json (4.3 MB)
   - web.json (4.3 MB)

### Verse Not Found?

If a verse doesn't appear, check:
- Book name spelling
- Chapter and verse numbers are valid
- Try the full book name (e.g., "John" not "Jn")
- Check browser console for errors (F12)
